# Asset Workspace

This folder is for production planning, references, and generated source material.

## Folder Rules

| Folder | Purpose |
|---|---|
| `art/` | Image, sprite, UI, concept, and visual reference resources |
| `audio/` | Sound, music, voice, and audio reference resources |

Runtime-ready files should go under `public/assets/`.

## Current Audio Status

No final audio files have been added yet. Audio folders are prepared so BGM and SFX can be kept separate from image resources.
