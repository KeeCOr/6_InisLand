# Audio Resources

Sound resources for planning and production.

| Folder | Purpose |
|---|---|
| `bgm/` | Music drafts, loops, and final BGM candidates |
| `sfx/` | Gameplay sound effects such as attacks, UI, building, zombie, weather, and gathering sounds |
| `reference/` | Audio references and mood samples |

Final runtime audio should be copied to `public/assets/audio/`.
