# InisLand — 눈보라 마을 기획서

> "끝없는 겨울. 마지막 마을의 마지막 수호자."

**장르:** 뱀서바이벌 + 낮/밤 베이스 디펜스 + 탑다운 생존  
**버전:** v0.2.0 · 2026-06-30  
**스택:** Unity 2D + C# + Resources SpriteBank + Windows Standalone

---

## 1. 문제 정의

"혼자서도 짧은 세션 안에 생존과 경영 두 재미를 모두 느낄 수 있는 게임이 없다."  
낮에는 설원 탐험가, 밤에는 마을 수호자 — 두 역할이 하나의 세션 안에서 자연스럽게 전환된다.

---

## 2. 페르소나

**주 페르소나**: 이진혁, 27세, 직장인  
- 자투리 시간(20~30분)에 게임. 긴 세션 불가.  
- 로그라이크/생존류 좋아하지만 복잡한 조작은 부담.  
- "오늘 밤은 버텼다"는 성취감을 원한다.

---

## 3. 코어 루프

```
낮(40s) → 설원 탐험 · 나무/돌/고기 채집 · 동물 사냥 · 동료 영입
   ↓ 저녁 자동 텔레포트
밤(40s) → 좀비 웨이브 격퇴 · 모닥불 방어 · 바리케이드 사수
   ↓ 새벽 회복: HP 복구 · 건물 +25%HP · NPC 1명 스폰
낮(다음 레벨)...
```

레벨 스케일: 낮/밤 페이즈 길이 = 40s × (1 + 0.25 × (레벨-1))

---

## 4. 주요 시스템

### 낮 페이즈
- **이동**: WASD, 설원 맵 자유 탐험
- **채집**: 나무(W), 돌(R), 사슴 사냥(공격)
- **영입**: 떠돌이 NPC 발견 시 대화로 동료 추가 (마을 House 수 한도)

### 밤 페이즈
- **좀비 웨이브**: 무한 증가, 25킬마다 보스 등장
- **보스 스프라이트**: `Assets/Resources/Sprites/Enemies/boss_frost_zombie.png`는 64x64 RGBA 단일 스프라이트로, 첫 보스 밤의 서리 좀비 보스에 사용
- **모닥불**: 2.5u DPS 화상 오라 + 마을 온도 유지 (연료 100s)
- **바리케이드**: 건물 HP 소진 시 마을 함락

### 건물 (10종)
| 건물 | 역할 | 비용 |
|-----|------|------|
| 🔥 Campfire | 좀비 화상 오라 + 온도 유지 | 5W |
| 🏠 House | 인구 한도 +4 | 6W |
| 🏹 Watchtower | 야간 자동 사격 | 8W+4S |
| 🌾 Farm | 식량 생산 | 6W |
| 📦 Storage | 자원 캡 +50 | 8W |
| 🥕 Fence | 외곽 차단 | 1W |

### 동료/NPC 타입
아이(빠름·약함·비전투), 노인(느림·농사+5), 사냥꾼, 치유사, 무사

### UI/HUD 목표 안내
- 좌측 하단 웨이브/스탠스 패널은 `상태 → 위험 → 다음 행동` 3줄 고정 슬롯으로 표시한다.
- 밤, 눈보라, 식량 부족처럼 판단이 필요한 상황은 위험 줄을 강조하고, 다음 행동 줄은 바로 수행 가능한 행동 문장으로 제한한다.
- 같은 HUD 패널 안에서 목표 안내, 인구, 동료 스탠스 버튼이 겹치지 않도록 300x118 고정 영역을 사용한다.

## 5. MVP 가설

| 가설 | 검증 방법 | 성공 기준 |
|------|---------|---------|
| 낮/밤 페이즈 전환이 자연스럽게 느껴지면 첫 세션에서 이탈 없이 3사이클 이상 플레이할 것이다 | 첫 플레이 관찰 | Day 3 생존 시도율 60%+ |
| 모닥불 방어가 핵심 판단 지점이 되면 밤 페이즈에서 긴장감이 올라갈 것이다 | 모닥불 주변 체류 시간·연료 관리 횟수 측정 | 밤당 연료 보충 행동 1회 이상 |
| 동료 영입이 낮 페이즈의 명확한 목표가 되면 탐험 동기가 유지될 것이다 | 낮 페이즈 이동 범위·NPC 발견 횟수 | 세션당 동료 영입 1인 이상 |
| 채집→건물→방어 인과가 명확하면 초보자도 튜토리얼 없이 첫 밤을 넘길 것이다 | 무튜토리얼 플레이 테스트 | 신규 플레이어 첫 밤 생존율 40%+ |

## 6. 레퍼런스 분석

| 레퍼런스 | 핵심 행동까지 단계 수 | 적용 교훈 |
|---------|-------------------|---------|
| Vampire Survivors | 시작→선택→첫 킬까지 3단계 | 첫 1분 내 성취(킬)를 주어 리텐션 확보 → InisLand에서 낮 첫 채집 10초 내 완료로 진입 장벽 낮춤 |
| Mincraft (서바이벌) | 스폰→첫 나무 채집→첫 도구까지 4단계 | 채집→제작→사용의 인과가 명확할수록 자발적 탐험 동기 생성 → 채집 자원이 건물로 즉시 연결되도록 설계 |
| Against the Storm | 낮/밤 사이클→자원 배분→생존까지 5단계 | 낮의 여유와 밤의 긴장이 극명히 대비될수록 밤 페이즈 몰입감 증가 → 낮 BGM/조명과 밤 BGM/조명 전환으로 페이즈 전환 명확화 |

## 7. 성공 KPI

- Day 3 생존율 50% 이상
- 세션당 평균 플레이 시간 15~25분
- 밤 1회 웨이브 격퇴 시 "성취감" 체감 (설문 7/10 이상)

---

## 8. 기술 스택

- **엔진**: Unity 2D
- **런타임 리소스**: `Resources/Sprites/...` + `SpriteBank`
- **패키징**: Windows Standalone BuildScript
- **테스트**: Unity EditMode/PlayMode 또는 가장 가까운 런타임 검증
