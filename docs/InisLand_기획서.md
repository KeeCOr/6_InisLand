## 문제 정의

**어떤 유저가:** 짧은 여가 시간에 전략적 판단을 즐기는 서바이벌 게임 팬  
**어떤 상황에서:** 모바일/PC 서바이벌 게임에서 "낮에는 채집, 밤에는 방어"라는 흐름을 반복하지만 선택이 단조롭게 느껴지는 상황  
**어떤 게임적 욕구를 원하는가:** "내 동료가 죽을 수 있다"는 무게감 있는 결정, 자원·배치·전투가 하나의 연결된 루프로 돌아가는 선명한 인과관계, 그리고 짧은 세션 안에서도 "이번엔 다른 전략을 쓰고 싶다"는 반복 동기

## 핵심 루프

```
[낮 — 설원 탐험]
  유저가 파티 구성과 출격 경로를 결정
    → 자원 수집 · 동물 사냥 · 동료 영입
      → 일몰 전 귀환 결정 (대담함 vs 안전)

[밤 — 마을 방어]
  귀환한 자원으로 바리케이드 · 건물 배치
    → 동료 스탠스 지정(따르기/사수/공세) + 무기·룬 조합
      → 좀비 무리 자동 공격 + 플레이어 직접 조작으로 새벽까지 생존

[세션 종료 — 보상/손실 정산]
  마을 번영도 상승 · 영속 성장 · 메타 해금
    → 영구 사망 동료 확인 → "다음 판엔 누구를 데려갈까" 결정
      → 다시 [낮 — 설원 탐험]
```

## 게임 소개

InisLand는 설원을 배경으로 한 2D 서바이벌 로그라이크다. **낮엔 탐험가**, **밤엔 성주**로 역할이 전환되며, 마을은 영속적으로 성장하지만 동료는 영구 사망한다. 이 비대칭 영속성이 모든 선택에 무게를 부여한다.

### 4가지 게임플레이 기둥

1. **낮의 사냥꾼** — 제한 시간 내 설원 탐험. 동료 영입, 자원 수집, 거대동물 사냥. 시야/안개가 긴장감을 만들고, 대담함과 귀환 타이밍을 저울질
2. **밤의 성주** — 모닥불로 불빛을 밝히고, 그리드에 바리케이드를 깔고, 영입한 동료들과 함께 좀비 무리를 새벽까지 막아냄. 뱀서류 자동 공격 + 타워디펜스 전략 배치
3. **영속하는 마을, 잃어버리는 동료들** — 마을은 계속 성장(영속). 하지만 동료는 영구 사망. "누구를 밖에 데리고 갈지"에 무게감 부여
4. **초대형 눈보라 (드문 이벤트)** — 좀비 + 극한 추위가 동시 발생. 모닥불/건물 밖에선 체온 대미지. 모든 시스템이 교차하는 절정 이벤트

## 주요 시스템

### 무기 카테고리 (6종)

| 무기 | 공격 패턴 | 특징 |
|------|----------|------|
| 롱소드 | 전방 휘두르기 | 균형, 초보 친화 |
| 창 | 전방 길게 찌르기 | 긴 사정거리, 관통 |
| 활 | 자동 조준 원거리 | 사거리 최장, 근거리 약함 |
| 서리 지팡이 | 광역 냉기 장판 | 슬로우 + 광역 |
| 워해머 | 강한 근접 슬램 | 기절, 저공속 |
| 쌍단검 | 고공속 근접 | 치명타 의존 |

### 진화 무기

기본 무기 + 특정 룬 2~3개 조합 충족 + 레벨 최대치 → 밤 중 자동 진화. 진화 효과는 해당 밤 종료 시 초기화 (기본 무기만 영속).  
예: 롱소드 + "연속 베기 룬" + "화염 룬" = **염화 폭풍검**

### 전체 세션 흐름

| 구간 | 내용 |
|------|------|
| Day 1~3 | 튜토리얼형 안정기, 기초 건물 해금 |
| Day 5 | 첫 보스 밤 |
| Day 7 | 초대형 눈보라 가능성 발생 |
| Day 10 | 두 번째 보스 + 동료 수 5~8 |
| Day 15+ | 눈보라 주기 점점 짧아짐, 좀비 Lv 상승 |
| Day 30+ | 엔드게임 (최고 난이도, 빌드 실험) |
| 마을 전멸 | 세션 종료, 점수 기록 + 메타 해금 |

## MVP 가설

| 번호 | 검증할 가설 | 검증 방법 |
|------|------------|----------|
| 1 | **낮→밤 역할 전환**이 있으면 유저가 한 세션 안에서 "탐험 플레이어"와 "방어 지휘관" 두 역할을 모두 경험했다고 느낄 것이다 | 1회 플레이 후 재시작률 60% 이상 |
| 2 | **동료 영구 사망**이 있으면 유저가 출격 파티 구성에 실제로 망설임을 경험하고, 사망 후 후회 감정이 재도전 동기로 이어질 것이다 | 동료 사망 이후 즉시 재시작 비율 50% 이상 |
| 3 | **위험·보상 동시 표시**가 있으면 낮 탐험 중 귀환 결정 평균 소요 시간이 8초 이하로 유지될 것이다 | 귀환 결정 평균 응답 시간 8초 이하, 결과 불만 피드백 20% 이하 |
| 4 | **초대형 눈보라 경고 신호**(종소리·까마귀·카운트다운)가 있으면 유저가 이벤트 시작 전에 준비 행동을 자발적으로 취할 것이다 | 눈보라 전 바리케이드 재배치 행동 비율 70% 이상 |
| 5 | **무기·룬 조합 진화**가 있으면 3판 내에 서로 다른 빌드를 시도하는 유저 비율이 50% 이상일 것이다 | 3판 내 상이 빌드 선택률 50% 이상 |

## 레퍼런스 분석

| 게임 | 장르 | 첫 재미 경험까지 UX 단계 수 | 핵심 교훈 |
|------|------|--------------------------|----------|
| Vampire Survivors | 뱀서류 로그라이크 | 2단계 (실행 → 자동 공격 시작) | 규칙 설명 없이 결과를 먼저 보여줌. 30초 이내 첫 레벨업 도파민 |
| Against the Storm | 생존 도시건설 | 4단계 (지도 선택 → 자원 확인 → 건물 배치 → 주민 반응) | 선택의 결과가 다음 턴에 즉시 가시화. "이번 판 실패 = 다음 판 학습" 명확 |
| Don't Starve Together | 서바이벌 탐험 | 3단계 (스폰 → 자원 채집 → 첫 밤 생존 시도) | 낮/밤 전환이 자연스럽게 긴장감을 형성, 별도 튜토리얼 없이 위험을 체감 |

**적용 교훈:** 튜토리얼 텍스트 최소화, Day 1 낮에 즉시 탐험 선택이 가능하게 구성. 밤 전환 30초 전 경고 신호로 방어 준비 행동을 자연스럽게 유도. 세션 종료 화면에서 "다음 판에 달라질 수 있는 것"을 바로 제안.

## 조작과 UX 원칙

- 주요 버튼은 44px 이상으로 유지하고, 화면당 CTA 강조색은 하나만 사용 (Von Restorff)
- 버튼/선택지는 한 번에 5개 이하로 노출해 판단 부담 최소화 (Hick's Law)
- 선택 후 400ms 이내 피드백 제공 (Doherty Threshold)
- HUD 동일 레이어 요소는 겹치지 않게 배치, 겹침 필요 시 별도 depth/z-order 사용
- 로딩 (Skeleton UI), 빈 상태 (아이콘+안내+CTA), 에러 (원인+해결+재시도), 많은 데이터, 긴 텍스트 — 5가지 상태 별도 처리

### 🖥️ UX / UI
- **낮↔밤 전환 타이밍 불명확:** 밤 전환 30초 전 화면 가장자리를 점진적으로 어둡게 물들이는 비네팅 효과 + 진행 바 추가. 텍스트 경고 없이 시각적 긴장감만으로 귀환 행동을 자연 유도 (Don't Starve Together 교훈 적용)
- **초대형 눈보라 진입 시 정보 과부하:** 눈보라 시작 순간 HUD를 "생존 3종(체온·모닥불 반경·동료 수)"만 남기고 비핵심 지표를 반투명 처리. 위기 상황에서 시선 집중 유도
- **세션 종료 화면 재도전 동기 부족:** 종료 화면에 "이번 판 Best 선택 vs 놓친 선택" 요약 카드 2장 제시. "다음 판 추천 빌드" 한 줄 텍스트와 함께 재시작 버튼을 강조색 CTA로 배치 (Von Restorff 원칙)
- **동료 초상화 구분 어려움:** 낮 탐험 파티 HUD에 동료 초상화 테두리 색상을 직업 유형별로 다르게 표시 (전투·지원·채집 3색 구분), 동일 레이어 내 겹침 없이 44px 이상 버튼 크기 유지

### 💰 수익화 전략
- **코스메틱 DLC (동료 스킨 팩):** 동료 3인(아주머니·아저씨·아이)의 대체 의상 스프라이트 팩. 영구 사망 시스템 특성상 유저가 동료에 감정적으로 투자하므로 외형 커스터마이징 구매 동기가 높음. 게임플레이에 영향 없어 밸런스 논란 없음
- **시즌 패스 (이벤트 챕터):** 분기별 신규 눈보라 시나리오 팩 (새 보스 1종 + 전용 룬 3종 + 한정 동료 1명). 한정 동료는 시즌 종료 후 구매 불가로 희소성 부여. 로그라이크 특성상 반복 플레이가 전제이므로 콘텐츠 추가가 직접적 플레이 시간 증가로 연결
- **사운드트랙 번들:** 낮/밤/눈보라 구간별 오리지널 OST. 장르 특성상 분위기 몰입도가 높아 사운드트랙 구매 수요 발생 가능

### 📦 콘텐츠 확장
- **단기(1~2개월):** 동료 3종 추가 (목사·의사·용병, 각각 고유 패시브 1개씩) + 낮 탐험 이벤트 카드 10종 (숨겨진 동굴·부상자 조우·상인 등) 추가로 탐험 변주 증가
- **중기(3~6개월):** "저주받은 유물" 시스템 도입 — 강력한 버프지만 동료 한 명에게 영구 디버프를 거는 선택지. 도덕적 딜레마를 핵심 루프에 추가. 또한 마을 건물 트리 3단계 확장(기초→중급→고급) 구현
- **장기(6개월+):** 협동 멀티플레이(2인) 모드 — 한 명은 낮 탐험, 다른 한 명은 밤 방어 배치를 동시 담당하는 비대칭 협동. 현재 낮/밤 역할 분리 기획이 자연스럽게 멀티 역할 분담으로 확장 가능

### ⚙️ 기술적 개선
- **Phaser 3 Object Pool 적용:** 좀비 대규모 스폰(Day 15+ 엔드게임) 시 동적 객체 생성·삭제 반복으로 GC 부하 급증 예상. `GameObjectFactory` 대신 `ObjectPool` 패턴으로 사전 할당 처리 → 엔드게임 프레임 드롭 방지
- **타일맵 청크 로딩:** 낮 탐험 맵이 커질수록 초기 로드 지연 발생 가능. 플레이어 이동 방향 기준 청크 단위 동적 로딩 적용 → 첫 탐험 진입 체감 속도 개선
- **세션 상태 자동 저장 (AutoSave):** 비정상 종료 시 Day 진행 상태 손실 방지. `localStorage` 또는 Tauri의 파일 시스템 API로 1분 주기 스냅샷 저장. 재진입 시 "이어하기/새 판" 선택 제공
- **룬·무기 조합 데이터 분리:** 현재 하드코딩 가능성이 있는 진화 조건 테이블을 JSON 설정 파일로 분리. 밸런스 패치 시 재빌드 없이 설정 변경만으로 반영 가능, 시즌 콘텐츠 추가 비용 절감
