# Runtime Assets

Game-ready assets used by the runtime belong here.

| Folder | Purpose |
|---|---|
| `characters/` | Player, companions, enemies, animals, and boss sprites |
| `buildings/` | Buildings, barricades, campfire, and defense structures |
| `props/` | Resource nodes, crates, logs, decorative objects |
| `tilesets/` | Ground and map tile sheets |
| `fx/` | Visual effects |
| `ui/` | HUD, icons, menus |
| `audio/` | Runtime BGM, SFX, and voice files |

Most current visual runtime candidates were imported from `C:\Development\0_KPM` because they are 6IL-related resources. Audio folders are prepared but no final audio files have been added yet.
