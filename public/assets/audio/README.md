# Runtime Audio Assets

Game-ready audio files belong here.

| Folder | Purpose |
|---|---|
| `bgm/` | Loopable music files |
| `sfx/` | Short gameplay and UI sound effects |
| `voice/` | Voice or vocal one-shot files, if used |

Preferred formats:

- `ogg` for runtime loops and SFX
- `wav` for source-quality exports when needed
